19 岁，是前端

有扎实的前端基础，熟练使用 HTML5, CSS3 进行页面布局, 熟练掌握 JavaScript 和 TypeScript 语言。

具备前端工程化能力，熟练使用 Node.js 脱离浏览器环境进行开发，掌握 cjs, es6 的模块封装，转换和兼容，熟练使用 npm, yarn, pnpm 进行依赖管理。

熟练掌握前端常用工具和组件库，如 Axios, Lodash, Unocss 等。

拥有前端架构能力，精通 Vue, 读过 Vue 源码，熟悉 Pinia, Vue-Router, VueUse 等工具。了解 React，并使用 Next.js 编写过全栈应用。

熟悉前后端分离的开发模式，了解 RESTful 接口风格进行数据请求，能够熟练掌握对接 Serverless 云函数。

掌握计算机基础技能，了解基本的数据结构和算法知识，了解密码学相关知识。

追求代码质量与设计，了解设计模式，遵守代码规范。掌握 OOP 和 FP 的思想。

有非 web 开发能力，熟悉 Python 并开发过爬虫，了解 Java, Go 语言。

了解服务器运维技能，熟悉 Linux/Unix 的基本操作和开发环境。

日常开发使用 Windows, 编辑器使用 VSCode 和 WebStorm。
